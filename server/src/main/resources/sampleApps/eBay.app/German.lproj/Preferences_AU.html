Verkäufereinstellungen verwalten

Tippen Sie unter „Einstellungen“ auf den Link „Verkäufereinstellungen“, um den gleichnamigen Bildschirm zu öffnen. In diesem Bildschirm können Sie die Rücknahmeinformationen eingeben, die Zahlungsmethoden festlegen und angeben, wie lange Sie für die Bearbeitung von Bestellungen benötigen. Auf diese Weise lassen sich Angebotsdetails, die für alle Ihre Angebote gelten, bequem vordefinieren.
